Sentiment-Driven Product Recommendation

This project utilizes how sentiment analysis can enhance product recommendations and sales insights within e-commerce platforms. It applies Natural Language Processing (NLP) techniques—including text preprocessing, feature engineering, and TF-IDF vectorization—to Amazon product reviews to develop a sentiment-driven recommendation system. Supervised machine learning algorithms, including Logistic Regression, Naïve Bayes, and Support Vector Machine (SVM), are used to classify user reviews into positive, negative, or neutral sentiments.
