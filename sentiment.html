Sentiment-Driven Product Recommendation

This project leverages NLP (TextBlob + TF-IDF) to analyze customer reviews and generate product recommendations. It uses a Flask API as backend and an interactive Chart.js/D3.js dashboard for data visualization.

Features

  • Visualize sentiment distribution of reviews
  • Backend API built with Flask
  • NLP pipeline with TextBlob & TF-IDF
  • Interactive charts with Chart.js and D3.js

This project utilizes how sentiment analysis can enhance product recommendations and sales insights within e-commerce platforms. It applies Natural Language Processing (NLP) techniques—including text preprocessing, feature engineering, and TF-IDF vectorization—to Amazon product reviews to develop a sentiment-driven recommendation system. Supervised machine learning algorithms, including Logistic Regression, Naïve Bayes, and Support Vector Machine (SVM), are used to classify user reviews into positive, negative, or neutral sentiments. The technology provides important data, including sentiment distribution, price-performance analysis, rating trends, and review patterns across product categories on an interactive visualization dashboard, in addition to producing individualized product suggestions. This two-pronged strategy helps customers and organizations to make more informed decisions by combining structured product criteria with unstructured user feedback.
